# 11ai AI Chat

Nine companion skills for building a grounded, tool-using AI chat and extending it across models, sessions, providers, interfaces, and messaging platforms.

## Choose a skill

| Skill | Use it for |
| --- | --- |
| [`11ai-ai-chat-stack`](./skills/11ai-ai-chat-stack/SKILL.md) | Building a complete Next.js App Router chat with the Vercel AI SDK |
| [`11ai-ai-chat-client-hooks`](./skills/11ai-ai-chat-client-hooks/SKILL.md) | Wiring `useChat`, restored history, streaming state, and persistence |
| [`11ai-ai-chat-session-mgmt`](./skills/11ai-ai-chat-session-mgmt/SKILL.md) | Adding session history, pinning, renaming, deletion, and storage |
| [`11ai-ai-chat-autotitle`](./skills/11ai-ai-chat-autotitle/SKILL.md) | Generating a new session title from the first user message |
| [`11ai-ai-chat-tool-design`](./skills/11ai-ai-chat-tool-design/SKILL.md) | Designing bounded, token-efficient tool sets grounded in application data |
| [`11ai-ai-chat-ui-ux`](./skills/11ai-ai-chat-ui-ux/SKILL.md) | Building and reviewing a polished, responsive chat interface |
| [`11ai-ai-chat-multiple-models`](./skills/11ai-ai-chat-multiple-models/SKILL.md) | Adding model selection, provider routing, and per-session persistence |
| [`11ai-ai-chat-github-provider`](./skills/11ai-ai-chat-github-provider/SKILL.md) | Using GitHub Models through its OpenAI-compatible endpoint |
| [`11ai-aichat-chatbot-extension`](./skills/11ai-aichat-chatbot-extension/SKILL.md) | Extending a web assistant to Slack or another messaging platform |

Start with `11ai-ai-chat-stack` for a new chat surface. Use a narrower companion skill when one layer already exists or needs focused work.
