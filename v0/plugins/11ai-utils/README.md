# 11ai Utilities

Eight general-purpose skills for code quality, text compression, reverse engineering, reviews, analytics, publication CMS, calls to action, and project sites.

## Choose a skill

| Skill | Use it for |
| --- | --- |
| [`11ai-code-quality`](./skills/11ai-code-quality/SKILL.md) | Reviewing and safely improving code clarity, structure, comments, maintainability, and regression coverage |
| [`11ai-compression`](./skills/11ai-compression/SKILL.md) | Shortening Markdown guidance locally while preserving its technical structure |
| [`11ai-platform-cta`](./skills/11ai-platform-cta/SKILL.md) | Adding context-aware platform calls to action with measurable conversion events |
| [`11ai-publications-cms`](./skills/11ai-publications-cms/SKILL.md) | Building a file-backed publications CMS with validated content and routes |
| [`11ai-reverse-engineering`](./skills/11ai-reverse-engineering/SKILL.md) | Producing a sanitized blueprint for rebuilding and modernizing a locally cloned repository |
| [`11ai-roast`](./skills/11ai-roast/SKILL.md) | Giving a blunt, prioritized, read-only critique of code or another work product |
| [`11ai-vercel-analytics`](./skills/11ai-vercel-analytics/SKILL.md) | Adding or verifying Vercel Web Analytics in a web application |
| [`11ai-www`](./skills/11ai-www/SKILL.md) | Examining a project and building a repo-driven site that displays it, in a `www/` folder or embedded app |

The compression skill includes local scripts for transforming and validating Markdown. The roast skill is read-only. For LLM usage and cost reporting, use the standalone [`11ai-llm-costs`](../11ai-llm-costs/) plugin. For repeated repository-wide audits of documentation, metadata, performance, security, or UX, use the [`11ai-super`](../11ai-super/README.md) plugin.
