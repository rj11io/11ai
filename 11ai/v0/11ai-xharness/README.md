# 11ai Cross-Harness

Two skills for routing and delegating work across AI agents running in different CLI harnesses.

## Choose a skill

| Skill | Use it for |
| --- | --- |
| [`11ai-xharness-agent-delegation`](./11ai-xharness-agent-delegation/SKILL.md) | Choosing a model and fallback tier for a delegated task |
| [`11ai-xharness-agent-comms`](./11ai-xharness-agent-comms/SKILL.md) | Calling, spawning, or continuing an agent in Claude Code, Codex CLI, or Gemini CLI |

Use delegation first when the receiving model is not already decided. Use agent comms to perform the handoff and collect the result.
