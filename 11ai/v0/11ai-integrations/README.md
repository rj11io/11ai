# 11ai Integrations

Three skills for adding Convex data and WorkOS AuthKit authentication to Next.js applications.

## Choose a skill

| Skill | Use it for |
| --- | --- |
| [`11ai-convex-integration`](./11ai-convex-integration/SKILL.md) | Adding Convex providers, backend functions, environments, and deployment setup |
| [`11ai-workos-integration`](./11ai-workos-integration/SKILL.md) | Adding WorkOS AuthKit routes, providers, protected pages, and environment setup |
| [`11ai-workos-convex-integration`](./11ai-workos-convex-integration/SKILL.md) | Configuring WorkOS and Convex together for an authenticated application |

Use the combined skill when the application needs both services; it covers the environment and provider interactions that the standalone skills cannot describe in isolation.
